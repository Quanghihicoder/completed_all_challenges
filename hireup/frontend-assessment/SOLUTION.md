## üíª Getting Started without Docker

### 1. Start the backend:

```bash
cd backend
npm install
npm start
```

> The backend server will run on [http://localhost:1337](http://localhost:1337)

```bash
cd frontend

```

> The frontend will run on [http://localhost:5173](http://localhost:5173)
